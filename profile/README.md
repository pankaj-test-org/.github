# pankaj-test-org

Testing organization for GitHub-Jenkins integration and CloudBees CI workflows.

## Repositories

**[marvels-mock](https://github.com/pankaj-test-org/marvels-mock)** (Public)  
CloudBees CI + GitHub integration testing. Validates ReRunCause for CBP-31531.

**[potential-train](https://github.com/pankaj-test-org/potential-train)** (Private)  
Mock APIs for Jenkins integration testing.

## What We Test

- GitHub-Jenkins integration (webhooks, branch indexing, build triggers)
- CloudBees CI features (GitHub Reporting plugin, build causes)
- CI/CD pipelines and GitHub Actions integration
- Configurable build failures via environment variables

## Quick Start

```bash
git clone https://github.com/pankaj-test-org/<repository-name>.git
```

Contact the organization owner for access to private repositories.
